This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.6.

## Source files

Source files located in 'source-files' folder.

# Dev mode

To run application in dev mode please clone the repo. Next, using terminal navigate to repo folder and than change to 'source-files' folder. Run 'npm install' to load all required modules. After that run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
